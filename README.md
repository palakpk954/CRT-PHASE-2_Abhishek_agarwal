# CRT Phase 2 — 16 Day Training

This repository contains all the code, notes, and problem solutions written during **Phase 2 of the Campus Recruitment Training (CRT)** program — a 16-day intensive focused on strengthening problem-solving and coding skills for placement preparation.

## 📌 About

Phase 2 builds on the fundamentals covered earlier and focuses on applying core concepts to solve interview-style problems consistently, one day at a time, over 16 days.

## 🗂️ Repository Structure

Code is organized day-wise, with each folder containing that day's problems and solutions.

